# ARTEM : Enhancing Large Language Model Agents with Spatial-Temporal Episodic Memory
Official repository for ARTEM (Agentic Retrieval with Temporal-Episodic Memory) — accepted as a poster at AAAI 2026. ARTEM is a hybrid LLM-based agent architecture integrating Spatial-Temporal Episodic Memory (STEM) for improved event encoding, storage, and retrieval across time.
