# TReMu-style Baseline

Status: runnable paper-inspired prompt baseline.

TReMu is not directly available as an installed package or local repo. This folder provides a lightweight implementation:

1. convert scoped events into dated timeline summaries;
2. retrieve timeline entries relevant to the query;
3. use temporal reasoning over the timeline to fill state slots.

This must be reported as `TReMu-style`, not as a faithful TReMu reproduction.
